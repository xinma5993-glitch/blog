# Repository Instructions

1. 本项目是部署于 GitHub Pages 项目路径 `/blog/` 的静态博客。
2. 使用原生 HTML、CSS 和 JavaScript。
3. 不引入构建步骤，除非用户明确同意。
4. 修改页面后必须检查相对路径，根目录页面和 `posts/` 文章页的资源深度不同。
5. 必须兼顾移动端、无障碍和深色模式。
6. 不提交密钥、Token、密码、`.env` 或个人敏感信息。
7. 提交前必须运行 `python scripts/check_site.py`。
8. 禁止 force push 和重写 `main` 历史。
9. 每次提交使用清晰的英文或中文提交信息。
10. README 与实际结构必须保持一致。
